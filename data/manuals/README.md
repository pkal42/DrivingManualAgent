# Sample Driving Manuals

This directory is for storing sample PDF driving manuals for testing purposes.

## Usage

1. Add sample driving manual PDFs to this directory
2. These PDFs will be used for testing the indexing pipeline
3. Sample PDFs can be uploaded to Azure Storage for processing

## Recommended Sample Documents

- State driving manuals (publicly available)
- Test PDFs with various formatting
- Documents with images and diagrams

## Note

Do not commit large PDF files directly to the repository. Use Git LFS if needed, or reference external storage locations.
